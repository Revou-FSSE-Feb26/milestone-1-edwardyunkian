# Software Engineer Portfolio Page — Edward Yunkian Janitra
---

## Overview
This is a personal portfolio page of Edward Yunkian Janitra as a beginner in Software Engineering. It is divided into 4 sections: home, about me, projects, and contact me section.

## Features
1. **Navigation**: at the top of page to quickly jump to different section of the page.
    * Click on the section you want to jump to.
    ![Navigation-1](/readme-img/navigation-1.png)
    * You will immediately jump to the coresponding section.
    ![Navigation-2](/readme-img/navigation-2.png)
2. **Home**: you can see my profile picture, brief introduction to my website, and the links to my socials. I also used one of my personal favorite picture as a background image.This section is made to be responsive using media queries in CSS, where the layout will adapt to various screen sizes to ensure the layout is still aesthetically appealing on different devices.
3. **About Me**: highlighting my work experiences, education, skills and interests. This section is also made responsive using media queries in CSS.
4. **Projects**: showcasing all the software engineering projects that I've done. Currently, there's still no project related to software engineering, but I still create a template to give an illustration how this section will be displayed. In the future, I plan to make the project cards interactive where it will expand and show the project details on click. But for now, I just add hover effect where the card's color will transition smoothly to pink when hovered.
![Project-Cards](/readme-img/projectcards.png)
5. **Contact Me**: inquiry form where you can send me your name and email, and send me a message directly.
